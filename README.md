# synwin
Synthetic Windows Dataset (and Generator)

# Usage
 1. Install blender3d
 2. Install ipython  
    ```
    path/to/blender/<version>/python/pip install ipython
    ```
 2. Set blender path
     ```
     export BLENDER_PATH=path/to/blender
     ```
 3. Run the script
    ```
    synwin.sh
    ```

# Debugging
 Run `bpy.sh`
